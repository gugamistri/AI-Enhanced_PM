## Glossário Terminológico

### Termos Mantidos em Inglês
**Justificativa:** Padrão estabelecido na indústria brasileira de tecnologia

- **Framework** - metodologia estruturada
- **Pipeline** - sequência de processamento
- **Deploy/Deployment** - implantação
- **Workflow** - fluxo de trabalho
- **Feedback** - retorno de informação
- **Insights** - percepções analíticas
- **Onboarding** - processo de integração
- **Sprint** - iteração de desenvolvimento
- **MVP** - Minimum Viable Product
- **API** - Application Programming Interface
- **DevOps** - Development Operations
- **MLOps** - Machine Learning Operations

### Termos Traduzidos Consistentemente
- **Feature** → **Funcionalidade**
- **Scaling** → **Escalamento**
- **Metrics** → **Métricas**
- **Testing** → **Testes** (substantivo), **Testar** (verbo)
- **Refactoring** → **Refatoração**
- **Governance** → **Governança**
- **Leadership** → **Liderança**
- **Revenue** → **Receita**
- **Market Share** → **Participação de mercado**

### Siglas Mantidas
- **IA** (não "Inteligência Artificial" por extenso)
- **KPI** - Key Performance Indicator
- **ROI** - Return on Investment
- **NPS** - Net Promoter Score
- **EBITDA** - Earnings Before Interest, Taxes, Depreciation, and Amortization
